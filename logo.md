# Logo Prompt

128x128px icon for a VS Code extension called "Unreal Scala LSP".

## Prompt

A minimal flat-design icon on a dark background. A translucent/ghostly Scala logo (the stylized "S" spiral staircase shape) rendered with a glitch effect — parts of the logo are slightly offset, fragmented, or fading out, suggesting something that looks real but isn't fully there. Use a cool color palette: electric blue (#4FC3F7) for the main shape with subtle purple (#B388FF) glitch artifacts. The background should be a dark charcoal (#1E1E1E) matching VS Code's dark theme. No text. Clean edges suitable for 128x128px rendering. The overall impression should be: "a Scala symbol that's almost real but slightly off."
